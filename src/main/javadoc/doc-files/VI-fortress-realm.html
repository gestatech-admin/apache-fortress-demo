Section VI - Install Fortress Realm

This section describes how to install The Fortress Realm Policy Enforcement Component

  1. Download the Apache Fortress Realm source bundle.

  2. Set JAVA_HOME: 
                    
    
# export JAVA_HOME=/opt/jdk1.7.0_10/
  3. Extract: 
                    
    
# tar xzf directory-fortress-realm-X.tar.gz
  4. Change directory: 
                    
    
# cd directory-fortress-realm/
  5. Set M2_HOME: 
                    
    
# export M2_HOME=../builder/apache-maven/

  6. Build fortress realm with maven and install to local repo (needed by demo app later on): 
                        
    
# $M2_HOME/bin/mvn clean install
