Section VII - Install Fortress Realm

This section describes how to install The Fortress Realm Policy Enforcement Component

  1. Clone the fortress realm project, from command line: 
                    
    
# git clone https://git-wip-us.apache.org/repos/asf/directory-fortress-realm.git
  2. Change into the project directory: 
                    
    
# cd directory-fortress-realm/

  3. Build fortress realm with maven and install to local repo. The fortress web [Section VIII - Install Fortress Web UI] and demo [Section IX - Install Fortress Demo] are both dependent on the fortress realm. 
                        
    
# mvn clean install
