Section IV - Install Fortress Core



This section provides instructions to install and configure Fortress & OpenLDAP and enable SSL traffic between them.

Steps to install the Fortress security system

  1. Perform [Section I - Instructions to extract and configure Fortress Builder Package to Target System] in FORTRESS APACHEDS QUICKSTART - Section I

  2. Perform [Section 2 - Instructions to run the Fortress Ant Build] in FORTRESS APACHEDS QUICKSTART - Section II

  3. Perform [Section III - Instructions to configure ApacheDS and load with seed data] in FORTRESS APACHEDS QUICKSTART - Section III

  4. Set M2_HOME: 
                    
    
# export M2_HOME=../builder/apache-maven/
  5. Build fortress core with maven and install to local repo (needed by demo app later on): 
                        
    
# $M2_HOME/bin/mvn clean install -DskipTests=true
