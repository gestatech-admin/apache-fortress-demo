Section IV - Install Apache Fortress



This section provides instructions to install and configure Fortress & OpenLDAP and enable SSL traffic between them.

Steps to install the Fortress security system

  1. Download the Fortress Demo #2 source bundle.

  2. Set JAVA_HOME: 
                    
    
                    # export JAVA_HOME=/opt/jdk1.7.0_10/
  3. Extract: 
                    
    
                    # $JAVA_HOME/bin/jar -xf apache-fortress-demo-master.zip
  4. Change directory: 
                    
    
                    # cd apache-fortress-demo-master/
  5. Set M2_HOME: 
                    
    
                    # export M2_HOME=../builder/apache-maven/
  6. Set ANT_HOME: 
                    
    
                    # export ANT_HOME=../builder/apache-ant/
  7. Build fortress core and install to local maven repo: 
                        
    
                        # $M2_HOME/bin/mvn clean install -DskipTests=true

  8. Perform [Section III - Instructions to configure ApacheDS and load with seed data] in FORTRESS APACHEDS QUICKSTART
